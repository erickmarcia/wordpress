Este plugin permite Agregar los Departamentos y Municipios de Nicaragua en la calculadora de envíos de su tienda WooCommerce.

== Description ==

Este plugin agrega a la calculadora de envíos, una lista desplegable de los Departamentos y Municipios de Nicaragua en su tienda de WooCommerce, 
esto proporciona una mejor experiencia para el usuario al momento de calcular el valor de los costos por envío.

Características:

 * Agrega a la calculadora de envíos, una lista desplegable de los Departamentos y Municipios de Nicaragua.
 * Agrega los Departamentos de Nicaragua a la pestaña de envíos en la página de administración de Woocommerce.
 * Agrega los Departamentos de Nicaragua a la pestaña de configuración en la administración de Woocommerce.

== Installation ==

= Minimum Requirements =

WordPress 4.0  o superior
Woocommerce 2.2 o superior
PHP version 5.2.4 o superior
MySQL version 5.0 o superior


= Automatic installation =

- La instalación automática es definitivamente la opción mas fácil, debido a que WordPress administra las actualizaciones de los archivos, simplemente inicie sesión en su panel de WordPress, navegue hasta el menú de plugins y haga clic en "Añadir Nuevo".
- Busque "Departamentos y Ciudades de Nicaragua para Woocommerce", instale y active.


= Manual installation =

Esta Sección describe como installar y activar el plugin de forma manual.

Pasos para la Instalación:

1. Descargue el plugin "Departamentos y Ciudades de Nicaragua para Woocommerce"
2. Transfiera el plugin al directorio `/wp-content/plugins/` de su Wordpress, Descomprima el archivo y deberá quedar `/wp-content/plugins/Departamentos y Ciudades de Nicaragua para Woocommerce`.
3. Navegue hasta el menú de plugins en su administración de WordPress, Busque "Departamentos y Ciudades de Nicaragua para Woocommerce" y presione clic en "Activar".
